## 🎯 规范要点概览

### VO和DTO使用规范
- **DTO**：用于数据传输，服务间调用、RPC接口、API请求响应
- **VO**：用于视图展示，Controller返回给前端的展示数据
- **命名规范**：`XxxDTO`、`XxxRequestDTO`、`XxxResponseDTO`、`XxxVO`
- **JSON注解**：必须使用`@JsonInclude`和`@JsonProperty`
