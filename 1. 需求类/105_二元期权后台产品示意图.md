## 2. 用户管理

```
+---------------------------------------------------------------+
| [搜索/筛选/分页]                                              |
+---------------------------------------------------------------+
| 用户列表（表格）：                                            |
| ┌──────┬───────┬───────┬───────┬───────┬───────┬───────┐      |
| │ 用户ID │ 昵称 │ 账户余额 │ 状态 │ 风控状态 │ 黑名单 │ 操作 │      |
| ├──────┼───────┼───────┼───────┼───────┼───────┼───────┤      |
| │ ...  │ ...   │ ...   │ ...   │ ...   │ ...   │ ...   │      |
| └──────┴───────┴───────┴───────┴───────┴───────┴───────┘      |
+---------------------------------------------------------------+
| [详情/编辑/风控/重置/黑名单标注等操作按钮]                   |
+---------------------------------------------------------------+
```

- **风控状态**：常用取值包括“正常”、“限额”、“冻结”、“观察”等。

## 6. 全局配置管理

```
+---------------------------------------------------------------+
| [配置项列表] [新增/编辑/删除]                                 |
+---------------------------------------------------------------+
| ┌───────────────┬─────────────┬───────┬───────┬───────┬───────┐|
| │ Key           │ Value       │ 说明  │ 修改人 │ 修改时间 │ 操作 │
| ├───────────────┼─────────────┼───────┼───────┼───────┼───────┤|
| │ ...           │ ...         │ ...   │ ...   │ ...   │ ...   │
| └───────────────┴─────────────┴───────┴───────┴───────┴───────┘|
+---------------------------------------------------------------+
| [全局配额/规则参数/操作日志/参数变更记录]                     |
+---------------------------------------------------------------+
```
- 支持全局风控限额配置（如全平台最大单笔/单日投注额等）。
- 支持用户级风控限额配置（如针对某用户的特殊限额）。
- **Key命名规则建议：**
  - 全局限额：`risk_limit.global.max_bet`、`risk_limit.global.max_daily_bet`
  - 用户限额：`risk_limit.user.{userId}.max_bet`、`risk_limit.user.{userId}.max_daily_bet`
- 说明字段中应明确key的作用和适用范围，便于后续维护和查找。
