# 工作计划整理

## Phase 1 (培训与集成)

### 关键项目
1. 市场营销功能
2. WL标准平台交易(WL Standard Platform deals) - 前端，可与BTSE TW共享技术团队
3. 许可交易所/合资平台
4. 二元期权 (API, 独立)

## Phase 2

### 关键项目
1. WL标准平台交易 - 独立前端，不依赖TW技术团队
2. 交易机器人 (全自动交易)
3. 监控平台 - 待确定 (前端)
4. API运维
5. 创新产品 (例如: 幻想加密货币)
6. WL移动应用 - 待确定
7. 预测市场 (如: Polymarkets)

## 关键问题与关注点

### 1. WL交易所和合规交易所功能升级
- **主要关注点**: 功能升级围绕市场营销功能展开
- **人力投入**: 与WL交易所数量相关（维护和新增）
- **关键问题**:
  - 是否有现有的产品计划或销售计划支持功能升级？
  - 是否有具体的时间表和优先级？

### 2. 二元期权的功能与运营
- **功能现状**: 交易品类和运营功能可能不完善
- **关键问题**:
  - 是否投入固定人力持续开发？
  - 是否根据运营情况灵活调整开发资源和功能需求？

### 3. 自动化交易的后续计划
- **关键问题**:
  - 自动化交易（交易机器人）的具体发展计划？
  - 是否有产品更新或新功能计划？（如策略优化、市场适应性调整等）

### 4. 其他新产品的计划与优先级
- **关键问题**:
  - 是否已有预测市场、幻想加密货币等创新产品的开发计划？
  - 这些新产品的开发优先级如何？
  - 是否会影响现有产品的开发进度？
