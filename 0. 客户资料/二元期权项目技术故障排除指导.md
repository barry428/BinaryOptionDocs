## 🔧 常用故障排除工具和方法

### 日志查看
```bash
# 查看服务日志
tail -f logs/option-service.log

# 查看特定时间段的日志
grep "2024-10-20 10:00" logs/option-service.log
```

### 服务状态检查
```bash
# 检查服务运行状态
ps aux | grep option

# 检查端口占用
netstat -tlnp | grep 8080
```

### 数据库连接检查
```bash
# 检查数据库连接
telnet localhost 5432
```
